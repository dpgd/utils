# Common libraries for TOP project


## Usage

### Install

#### The first user on the team:
* cd your-project-that-uses-utils-lib
* git submodule add git@10.25.253.3:TOP/utils.git

#### Other team's members
* cd your-project-that-uses-utils-lib
* git submodule init 

### Update
* git submodule update
