# Common libraries for Django Projects


## Usage

### Install

#### The first user on the team:
* cd your-project-that-uses-utils-lib
* git submodule add git@github.com:dpgd/utils.git

#### Other team's members
* cd your-project-that-uses-utils-lib
* git submodule init 

### Update
* git submodule update
* git pull
